# dotfiles

This is *my* simple system config for OSX.
Still a WIP. Go ahead and fork it at your own risks.

## Usage
Clone this repo in the $HOME folder
```
git clone http://github.com/llcoolju/dotfiles
cd dotfiles
./install.sh
```
